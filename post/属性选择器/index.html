CCS3 属性选择器

和jquery类似 C3中也有不少属性选择器了。

[att] 表示选择具有att属性的元素 att可以是自定义属性

1
2
3

li[index]{
    color:red;
}

[att~=“val”] att属性中有用空格分开的val属性

1
2
3

li[class~=fruit]{
    backgroundColor:"green";
}

[att^=“val”] att属性的值以val开头

1

li[class^="icon-"]{}

[att$=“val”] att属性的值以val结束

1

[class$="gg]{}

[att*=“val] att属性中包含字符串w

1
2
3

[class*="z"]{
    fontSize:20px;
}
